# Primer Evidencia

## **Pasos para la ejecucion del proyecto**

1 Clonar este repositorio.

2 Ingresar a la carpeta del proyecto clonada.

3 Instalar las dependencias necesarias (npm install).

4 Ejecutar la aplicacion con npm run dev.
